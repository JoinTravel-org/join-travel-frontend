# JoinTravel Frontend

## Technologies

- React
- TypeScript
- Vite
- Deno

## How to run

### Dependencies

Install the deno runtime.

```bash
# Using curl
curl -fsSL https://deno.land/install.sh | sh

# Using npm
npm install -g deno
```

Install the needed modules.

```bash
deno install
```

### Execute

```bash
# Start development environment using Vite
deno run dev

# Build app for production
deno run build

# Preview production app with Vite
deno run preview
```
